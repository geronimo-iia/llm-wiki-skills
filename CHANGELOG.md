# Changelog

## 0.1.0 — 2025-07-18

Initial release with 11 skills:

- **setup** — install llm-wiki and manage wiki spaces
- **bootstrap** — session orientation (config, types, hub pages)
- **ingest** — process source files into synthesized wiki pages
- **crystallize** — distil sessions into durable wiki pages
- **research** — search and synthesize from existing knowledge
- **lint** — structural audit (orphans, stubs, broken links)
- **graph** — generate and interpret the concept graph
- **frontmatter** — reference for writing correct frontmatter
- **skill** — find and activate skills stored in the wiki
- **write-page** — create a wiki page of any type
- **configure-hugo** — configure a wiki for Hugo rendering
